# Flight Attitude HUD Simulation (C / Raylib)

A real-time flight attitude simulator written in C using Raylib.

This project implements a fighter-style Head-Up Display (HUD) with an artificial horizon, pitch ladder, and flight marker. The simulation demonstrates basic aircraft attitude dynamics and interactive control.

## Features

- Artificial horizon rendering
- Pitch ladder display
- 360° roll and pitch dynamics
- Inverted flight control behavior
- Dynamic altitude display
- Real-time keyboard control
- Fighter-style HUD interface

## Controls

W / S → Pitch control  
A / D → Roll control  
ESC → Exit simulation

## Technologies

- C
- Raylib graphics library
- Real-time simulation loop

## Concepts Demonstrated

- Flight attitude visualization
- Coordinate transformations
- Real-time rendering
- Basic flight dynamics
- Interactive simulation

Compile using GCC:

## Run

./sim


## Project Goal

The goal of this project is to simulate key elements of a fighter aircraft HUD and demonstrate real-time flight attitude visualization using low-level C programming.
